== Description ==

Simply Materialized Portfolio is a modern WordPress theme inspired by Material Design which can be used as a template for a variety of websites.

**Current Features:**
* Utilizes the Materialize framework
* Front End Publishing of Blog Posts
* Parallax scrolling featured images and blog images

**Coming Soon:**
* Shortcodes

== Installation ==

= Using WordPress' Theme Installer =

1. In the Admin area, click on `Appearance => Themes => Add New => Upload Theme`
2. Click `Browse...` and select the `simply-materialized-portfolio.zip` file
3. Click `Install Now`
4. Activate the plugin through the `Appearance => Themes` Menu

= Manual Installation =

1. Unzip the `simply-materialized-portfolio.zip` file
2. Upload the resulting folder to the `/wp-content/themes` directory
3. Activate the plugin through the `Appearance => Themes` Menu
